Custom Fields

Add custom fields to the request form. Format as a JSON array:
[{"name": "Priority", "placeholder": "High/Medium/Low", "required": false}]

To add multiple fields, separate them with commas inside the array:
[{"name": "Field1", "placeholder": "...", "required": true}, {"name": "Field2", "placeholder": "...", "required": false}]
